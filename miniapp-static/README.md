# Telegram Mini App Static Site

Эта папка подготовлена для отдельного деплоя как `Render Static Site`.

Что публиковать:

- `Publish Directory`: `miniapp-static`
- `Build Command`: оставить пустым

После деплоя Render даст URL вида:

- `https://<your-static-site>.onrender.com`

Именно этот адрес потом можно будет поставить в `MINI_APP_URL` у бота.

Текущие файлы:

- `index.html` — standalone mini app
- `assets/cards/` — изображения карт
